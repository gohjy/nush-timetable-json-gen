Guide to Course Code Input
  • Use the + button to add more than one course code (e.g. MTL options)
  • A blank field indicates a free period or one that is left blank on the timetable
  • Leave a blank field at the end of a list for elective slots and other slots where not everyone has a class
  • RC => recess
  • LUN => lunch
  • MEN => mentoring
  • CCE => CCE
  • PE => PE
  • CCA => CCA
